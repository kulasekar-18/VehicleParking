# 🚗 Vehicle Parking Management System

A full-stack Vehicle Parking Management System built using **React, Node.js, Express, and MongoDB**.

This project manages parking slots, bookings, employees, and revenue with role-based access.

---

## ✨ Features

### 👤 User

- View available parking slots
- Lock a slot temporarily
- Book a parking slot
- View booking history

### 👷 Employee

- Employee dashboard
- Start / End shifts
- Vehicle entry & exit
- Activity tracking

### 🛠 Admin

- Admin dashboard
- Create / update / delete parking slots
- Manage employees
- View active employee shifts
- Revenue & activity reports
